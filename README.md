# おんぷメーカー 🎵

たべものなどの ことばを マイクで はなすと、リズムボックスにのせて おんぷ表示してくれる、こども向けの音楽教育アプリです。

**例**
- 「バナナ」 → バ・ナ・ナ（♩♩♩）
- 「スパゲッティ」 → ス・パ・ゲ・ッ・ティ（♪♪♪𝄽♪）

## 主な機能

- 🎤 **音声入力**：日本語をマイクで認識（Web Speech API）
- 🎼 **モーラ分割**：日本語の拍（モーラ）単位で自動的に音符化
  - 拗音（ャュョ）は直前の文字と結合
  - 促音（ッ）・長音（ー）・撥音（ン）は独立した1モーラ
- 🥁 **リズムボックス**：10種類のドラムパターン（シンプル / ロック / ディスコ / マーチ / ワルツ / サンバ / ボサノバ / ヒップホップ / シャッフル / レゲエ）
- 🎚 **スピード調整**：50〜200 BPMをスライダーで調整
- 🎹 **メロディ再生**：認識した言葉を音符として順に演奏（母音ごとに音程が変化）
- 🍎 **プリセット**：マイクが使えない環境用に、たべものの言葉ボタンを用意

## 使い方

1. **リズムボックス**でパターンと速さを選び、▶ボタンで再生開始
2. **🎤 おしてはなす**を押して、たべものの名前を声で言う（または下のプリセットから選ぶ）
3. **おんぷ**エリアに音符が表示されたら、▶ おんぷを ならす で再生

## 動作環境

- Google Chrome（マイク機能のため推奨）
- Microsoft Edge も可
- Firefox / Safari ではマイクが使えない場合があります（プリセットボタンは使えます）

ローカルで動かす場合は、このフォルダを直接 `index.html` で開くだけでOKです（外部依存ゼロ）。
